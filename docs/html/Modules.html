API documentation for modules¶

This document contains the help documentation for the modules and is intended for users who will be writing/using the python code itself. For everyone else, please look at the Scripts documentation.

NumSwitch.py¶

NumSwitch.NS(Ainv, eps, k, l)[source]¶

This function implements equation 3.5: gives the number of sign switches when perturbing the k, l entry

Parameters:
  • Ainv – inverse input matrix, numpy array
  • eps – perturbation size (scalar)
  • k – index
  • l – inex
Returns:

Natural number
NumSwitch.critical_epsilon(Ainv, k, l, i, j)[source]¶

This function finds which epsilon causes the sherman morrison ratio to equal 0 (indicating a switch will occur after this value). Takes advantage of the monotonicity of the SM ratio.

Parameters:
  • Ainv – Inverse input matrix
  • k – index (row being perturbed)
  • l – index (column being perturbed)
  • i – index (entry looking for a switch)
  • j – index (entry looking for a switch)
Returns:

scalar (epsilon value)
NumSwitch.exp_num_switch(A, Ainv, k, l, num_sample=1000, dist=None, interval=None)[source]¶

This implements equation 3.6: the expected number of sign switches

Parameters:
  • A – The original input matrix (numpy array)
  • Ainv – The inverse of A (pre-computed), numpy array
  • k – index
  • l – index
  • num_sample – Number of samples to take to determine the interval of stability and expected value
  • dist – A probability density function: function of a single argument
  • interval – the interval of stability (optional, can be pre-computed with interval_of_stability())
Returns:

NumSwitch.interval_of_stability(A, Ainv, k, l, max_bound=10)[source]¶

This function will return the interval of stability when perturbing just the (k,l) entry of A

Parameters:
  • A – numpy array
  • Ainv – inverse of A
  • k – index
  • l – index
  • max_bound – some are always stable on one side, only go out this far), scalar > 0
  • num_sample – number of points (natural number) to sample to determine stability
Returns:

(a:scalar, b:scalar) the largest interval where A[k,l]+eps is stable
