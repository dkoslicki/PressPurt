Command line scripts¶

This describes how to interact with the command line scripts.

PreprocessMatrix.py¶

This script pre-processes a matrix by figuring out what the intervals of asymptotic stability are, as well as finding which perturbation values lead to a sign switch.

usage: PreprocessMatrix.py [-h] [-p PREFIX] [-m MAX_BOUND] [-z] [-t THREADS]
                           input_file output_folder

Positional Arguments¶

input_file Input comma separated file for the jacobian matrix.
output_folder Output folder. A number of files will be created in the form ‘output_folder/<prefix>_*.npy’

Named Arguments¶

-p, --prefix Prefix of output files, if you so choose.
-m, --max_bound
 

some of the matrices are unbounded stable towards one end, this is the limit the user imposes

Default: 10

-z, --zero_perturb
 

Flag to indicated you want to pertub the zero entries.

Default: False

-t, --threads

Number of threads to use.

Default: 8

ComputeEntryWisePerturbationExpectation.py¶

This script computes the expected number of sign switches from perturbing each entry individually

usage: ComputeEntryWisePerturbationExpectation.py [-h] [-p PREFIX]
                                                  [-d DISTRIBUTION_TYPE]
                                                  [-a A] [-b B] [-t THREADS]
                                                  input_folder

Positional Arguments¶

input_folder Input folder. The location of the files created by PreProcessMatrix.py. eg ‘output_folder/<prefix>_asymptotic_stability.npy’. This is also where the expected num swith array will be saved.

Named Arguments¶

-p, --prefix Prefix of output files, if you so choose.
-d, --distribution_type
 

Kind of distribution to use. Valid choices are: truncnorm, uniform, trunc_lognorm, beta.

Default: “truncnorm”

-a

First parameter to the distribution you choose. For truncnorm, this is the mean.

Default: 0

-b

First parameter to the distribution you choose. For truncnorm, this is the variance. Using a negative value indicates you want the standard deviation to be the length of the interval divided by the absolute value of the input parameter.

Default: -2

-t, --threads

Number of threads to use.

Default: 8

GenerateEntryWiseFigures.py¶

This script computes the expected number of sign switches from perturbing each entry individually

usage: GenerateEntryWiseFigures.py [-h] [-p PREFIX] [-a]
                                   [-l LIST_OF_NUMSWITCH_TO_PLOT [LIST_OF_NUMSWITCH_TO_PLOT ...]]
                                   input_folder

Positional Arguments¶

input_folder Input folder. The location of the files created by PreProcessMatrix.py. eg ‘output_folder/<prefix>_asymptotic_stability.npy’. This is also where the expected num swith array will be saved.

Named Arguments¶

-p, --prefix Prefix of output files, if you so choose.
-a, --all_numswitch_plots
 

Include this flag if you want all the num switch plots (it could be large)

Default: False

-l, --list_of_numswitch_to_plot
 List of entries you want visualized with num switch. Eg. -l 1 1 1 2 to plot the (1,1) and (1,2) entries.

ComputeQuantitativeSensitivity.py¶

This script Generates the quantitative sensitivity: perturbing each single entry off to infinity.

usage: ComputeQuantitativeSensitivity.py [-h] [-p PREFIX]
                                         input_file output_folder

Positional Arguments¶

input_file Input comma separated file for the jacobian matrix.
output_folder Output folder. A file <prefix>_quantitative_sensitivity.csv’ will be put here.

Named Arguments¶

-p, --prefix Prefix of output files, if you so choose.

GenerateQuantitativeSensitivityFigure.py¶

This script Generates the quantitative sensitivity figure.

usage: GenerateQuantitativeSensitivityFigure.py [-h] [-p PREFIX] input_folder

Positional Arguments¶

input_folder Input folder containing the output of ComputeQuantitativeSensitivity.py

Named Arguments¶

-p, --prefix Prefix of output files, if you so choose.

ComputeMultiEntryPerturbationExpectation.py¶

This script pre-processes a matrix by figuring out what the intervals of asymptotic stability are, as well as finding which perturbation values lead to a sign switch.

usage: ComputeMultiEntryPerturbationExpectation.py [-h] [-n NUM_ITERATES]
                                                   [-l INTERVAL_LENGTH]
                                                   [-t THREADS]
                                                   input_file

Positional Arguments¶

input_file Input comma separated file for the jacobian matrix.

Named Arguments¶

-n, --num_iterates
 

Number of iterates in the Monte Carlo sampling to perform.

Default: 10000

-l, --interval_length
 

Interval length over which to make the perturbations.

Default: 0.01

-t, --threads

Number of threads to use.

Default: 8
